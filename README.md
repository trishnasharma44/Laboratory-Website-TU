# Laboratory-Website-TU
A Sensors, Measurements and Instrumentation Research laboratory website developed for Tezpur University to provide information about laboratory facilities, equipment, and academic activities in a structured and user-friendly manner.

- Laboratory Overview
- Available Resources
- Projects
- Publications
- Faculty, Staff and Students Details
- Contact Information

# Technologies Used
- React.js
- Tailwind CSS

# Usage 
Open the website in any web browser to view laboratory details and related information.
This project was developed as part of my 7th Semester B.Tech academic work.
